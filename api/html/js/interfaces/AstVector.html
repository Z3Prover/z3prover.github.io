Interface AstVector<Name, Item>

Stores multiple Ast objects

const vector = new AstVector<Bool>();
vector.push(Bool.val(5));
vector.push(Bool.const('x'))

vector.length
// 2
vector.get(1)
// x
[...vector.values()]
// [2, x]
