Interface AstMap<Name, Key, Value>

Stores a mapping between different Ast objects

const map = new Map<Arith, Bool>();
const x = Int.const('x')
const y = Int.const('y')
map.set(x, Bool.val(true))
map.Set(y, Bool.val(false))

map.size
// 2
map.has(x)
// true
[...map.entries()]
// [[x, true], [y, false]]
map.clear()
map.size
// 0
